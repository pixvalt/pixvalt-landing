Nothing to hide about how we protect you.

We publish exactly what encryption we use and how it works. No vague "military-grade" promises.

Algorithm

AES-256-CBC

The same encryption standard used by banks, governments, and the US military. 256-bit key — mathematically unbreakable by brute force.

Key Derivation

PBKDF2-HMAC-SHA256

Your 6-digit PIN is stretched into a 32-byte encryption key using 100,000 iterations. Slow by design — makes brute force attacks impractical.

File Format

.pvlt (PixValt)

Every encrypted file is wrapped in our proprietary .pvlt format with a random 16-byte IV per file. No other app on Earth can open a .pvlt file.

Key Storage

In-Memory Only

Your derived key lives in memory only while the vault is unlocked. It is never written to disk, never sent anywhere, and wiped on app lock.

Server Access

Zero

PixValt has no servers that receive your files. Drive backups go directly to your personal Google Drive account — not ours.

App Protection

Play Integrity API

PixValt detects tampered or repackaged APKs using Google's Play Integrity API. Fake versions of PixValt cannot access your vault.
